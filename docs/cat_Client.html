Category Client

Description:

Functions related to the state of the client struct/object, or to its state.
This includes logging in and out, and any settings for SDK logging.

Related Functions:
ClearCredentials() Clears credentials set by a previous call to SetCredentials().
HasCredentials() Indicates whether credentials have been set by a previous call to SetCredentials().
IsLoggedIn() Indicates whether the client has successfully invoked Login().
Login() Establishes a session with the director, using the given credentials.
Subsequent REST API invocations will be accompanied by a security token which the director associates with this session.
LogOut() Clears session information created during a previous invocation of Login().
SetCredentials() Establishes credentials to be used for all subsequent REST API invocations.
